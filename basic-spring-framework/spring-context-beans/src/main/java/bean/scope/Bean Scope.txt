定义和声明bean scope: 支持XML和Annotation两种形式
1. singleton
   <bean id="loginService" class="***.LoginService" scope="singleton" />

   @Bean("loginService") 支持定义bean的别名,可以使用类型的全路径
   @Scope("singleton")

2. prototype
   <bean id="loginService" class="***.LoginService" scope="prototype" />

   @Bean({"org.springframework.security.config.**.httpSecurity"})
   @Scope("prototype")
