TODO. @EnableAspectJAutoProxy(proxyTargetClass = false, exposeProxy = false)
本质上是往Spring IoC中注入了一个后置处理器, 生成代理对象并添加对应的Aspect切面
AnnotationAwareAspectJAutoProxyCreator.class extends BeanPostProcessor

- proxyTargetClass = false Jdk动态代理(实现接口)或Cglib代理(继承子类)
- exposeProxy = false 默认不将Proxy暴露给ThreadLocal, 无法通过AopContext.currentProxy()获取
- exposeProxy=true;   从线程变量中取出代理对象调用方法, 等效造成Aspect切面效果
